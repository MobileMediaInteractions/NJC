# First-party entitlement

First-party entitlement is an ordinary signed `first_party` license bound to exact registered application identities. It is not a hardcoded unlimited bypass. The vertical slice proves the New Jersey Courier web identity succeeds while another host fails.

Production issuance still requires a trusted identity assertion: Apple App Attest/DeviceCheck, Android Play Integrity, a web deployment assertion, or an equivalent server-controlled release identity. No public issuance endpoint exists until that provider is selected. The current web showcase is development-only and production returns 404.
