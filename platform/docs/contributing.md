# Contributing

Preserve trust boundaries and unrelated application behavior. Update grammar, formatter, semantic model, binary schema, runtime, docs and tests together. Add corruption/limit tests for every reader change and exact authorization tests for every licensing change. Never add a production development bypass.

Run `pnpm platform:check`, `pnpm platform:demo`, `pnpm platform:benchmark`, the C ABI smoke compile, web/employee checks, and the playground build. Record architecture/security decisions in an ADR before adopting a new renderer, serializer, crypto primitive or executable-module system.
